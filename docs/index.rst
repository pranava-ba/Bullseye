====================================
Bullseye Fintech — NSE EOD Dashboard
====================================

.. image:: https://img.shields.io/badge/version-2.0.6-amber
   :alt: Version 2.0.6

.. image:: https://img.shields.io/badge/platform-Windows%2010%2F11-blue
   :alt: Windows 10/11

.. image:: https://img.shields.io/badge/python-3.10%2B-blue
   :alt: Python 3.10+

----

**Bullseye Fintech NSE EOD Dashboard** is a Windows desktop application
built on Streamlit that downloads, processes, and visualises NSE (National
Stock Exchange of India) end-of-day bhavcopy data.  It provides interactive
OHLCV charts, a consolidated four-method pivot point matrix, and clean CSV
exports — all without requiring a database or cloud connection.

.. tip::

   New here?  Jump straight to the :ref:`quickstart` to download your first
   year of NSE data and see a chart in under five minutes.
